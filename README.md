# 🌦️ Weather App

A simple Python-based Weather Application that fetches real-time weather data using the OpenWeather API.

---

## 🚀 Features

- 🌍 Search weather by city name
- 🌡️ Displays temperature, humidity, and pressure
- 💨 Shows wind speed
- ☁️ Weather condition description
- 🖥️ Clean CLI (Command Line Interface) UI
- 🔁 Continuous input loop for multiple searches

---

## 🛠️ Technologies Used

- Python
- Requests Library
- OpenWeather API
